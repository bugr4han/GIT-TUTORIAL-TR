<h1 align="center">Git & GitHub</h1>

<p align="center">
Bu depo, Git ve GitHub’ı temelden öğrenmek isteyen kullanıcılar için kapsamlı bir kaynak olarak hazırlanmıştır.  
İçerikte; Git’in temel prensipleri, versiyon kontrol süreçleri, komutlar, GitHub hesabı oluşturma, GitHub CLI kullanımı ve depo yönetimi gibi birçok konu <strong>adım adım</strong> ele alınmaktadır.
</p>

Bu eğitim sayesinde:

- Projelerinizi Git ile versiyonlayabilir,
- GitHub üzerinde depolar oluşturabilir ve yönetebilir,
- Branch yapılarıyla daha düzenli çalışabilir,
- Terminal üzerinden GitHub CLI ile gelişmiş işlemler yapabilir hale gelirsiniz.
